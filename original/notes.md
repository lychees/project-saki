## 对话系统

- 10 个对话框样式（`dialogList.json`）：1=无头像框、2=图片头像框、3=立绘头像框、4/5/6=无对话框(下/中/上)、7/8=大段文本(NVL)、9=未命名（章节标题卡用，配合打字机音效 `se/打字.mp3`）
- 说话人由「名字字段」或「立绘头像 ID」确定：2=Kotori、3=OnePlus、4=MoonMayCry、5=xiaodai、6=sevenzero、7=幼Kotori
- 其他具名 NPC：母上、父亲、班主任、？？？
- 文本为 HTML（`<span style='color:#FfFffF'>…`），绝大多数是纯白色
- 特殊标记：
  - `[p5]` / `[p15]` / `[p50]`：对话内停顿/分页标记（数值越大停越久），重制时映射为 Ren'Py 的 `{w}` / 换页
  - `[$vN]` `[$wN]` `[$sN]`：数值/世界/字符串变量插值；`[@vN]` `[@wN]` `[@sN]` `[@pN]` `[@fN]`：扩展插值（p=指令参数，f=自定义函数）
- 每条对话可带「表情/语气编号」（改变头像框中立绘表情）和语音（本作未使用语音）

## 数据格式要点（GameCreator 0.98 发布版）

- `asset/json/startup.json` 是**打包清单**：key=相对路径，value=该数据文件完整内容（UI、对话框样式、自定义指令定义等）；场景数据不在其中，需按 `asset/json/scene/data/scene<N>.json` 与 `asset/json/server/scene/s<N>.json` 按需下载
- 剧情 = `s<N>.json` 的 `customCommands[i]` 指令数组，每条指令为 `[指令ID, 参数…, {___cmdID}]`
- 核心指令 ID：0/1/2=如果/否则/结束，3/4/10=选项开始/选项项/选项结束，5/6/7=循环开始/结束(跳回)/跳出，8/9=标签/跳转，11=显示对话 `[对话框ID,立绘头像ID,名字,速度,对象,HTML文本,语音,表情,名字颜色]`，12=等待输入，13=变量操作，16=调用公共事件，17=等待(帧/ms)，18=注释，20=执行JS，21=切换场景，63/64=播放/停止BGM，65/66=环境音，67=SE，69=语音
- 自定义指令（ID≥10000，定义在 `customCommandType.json`，参数字典的 key 是属性 ID）：10001=预加载、10007=解锁CG、40001=显示图片、40002=移动图片、40007=移动立绘、40008=消除图像、40010=显示界面、40012=关闭界面、40014=添加材质、40017=更换场景、50005=开始游戏、50008=返回标题、160001=双指缩放、160002=图像震动
- 立绘：`standAvatar/data/standAvatar<N>.json` 的 `picUrls`（一个角色多张差分图，按帧索引切换表情）
- 原始数据完整备份在 `original/raw/`（含 startup.json 解包出的 151 个 JSON + 后续按需下载的场景/立绘数据）
